
# Andamento del numero di avvisi inviati

Il file riporta il numero di avvisi inviati tramite App IO o altri canali, per notifiche SEND da leggere. Il dato è sugli ultimi 3 anni e un valore complessivo su tutto lo storico.

## Aggiornamento dati

- Quotidiano, alle 11:00 (UTC).

## Formato dati

**Reference file:**

- messaggi_di_cortesia.csv<br>

| Campo                         | Tipo di dati  | Descrizione                                                       | Formato |
| ----------------------------- | ------------  | ----------------------------------------------------------------- | ------- |
| anno                          | string        | Anno di invio degli avvisi                                        | YYYY    |
| messaggi_di_cortesia_inviati  | integer       | Numero di avvisi inviati                                          |         |
| canale_di_comunicazione       | string        | Tipologia di canale tramite cui sono stati inviati i messaggi     |         |

Questi dati sono disponibili anche in formato json.
